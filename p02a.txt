１）forループの記述方法と，実行がどのように行われるか，例を示して説明しなさい。
　　ただし，for文の3つの要素A,B,Cについて，その役割と実行されるタイミングについての説明を含むようにしなさい。
　　for ( 要素A ; 要素B ; 要素C )
要素A：ループの初期化: ループの最初に一度だけ実行される．
要素B：ループの継続条件: ループの継続条件が真の間ループが繰り返される．
要素C：ループの更新: ループの各繰り返しの終了時に実行される．

２）whileループの記述方法と，実行がどのように行われるか，例を示して説明しなさい。
while( 条件式 ) {
  条件式が真の間実行される処理
}
プログラミングあるいはループ起動の時点では未定．条件式が真の間，処理を繰り返す．
例: 奇数になるまで2で割るプログラム．
int n;
scanf("%d", &n);
while ( n % 2 == 0 ) {
  n = n / 2;
}
printf("%d", n);

３）do-whileループの記述方法と，実行がどのように行われるか，例を示して説明しなさい。
do {
  条件式が真の間実行される処理
} while ( 条件式 );
条件式が真の間，処理を繰り返す．プログラミングあるいはループ起動の時点では未定．少なくとも1回は処理を行なう．
例: 入力された数を表示するプログラム．負の数が入力されるまで表示させない．
do {
  scanf("%d", &n);
} while ( n >= 0 );
printf("%d", n);


４）whileループとdo-whileループの違いを説明しなさい。
必ず1回は処理を行なうものはdo-whileループが使われる．
1回も処理が行なわれない可能性があるものはwhileループが使われる．

５）字下げのルールについて，中かっことの関係で説明しなさい。
中かっこの中に書かれた処理は，中かっこの前にある処理の中に入る．わかりやすくするために中かっこ内の処理は字下げする．
